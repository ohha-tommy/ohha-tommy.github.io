リストの書き方（ul+li、ol+li）

リストは ul と li タグを使いやす。

  • HTML
  • CSS
  • JavaScript
  • チョコクランチ

ul の代わりに ol を使うと箇条書きの頭に連番がつきやす

  2. CSS
  3. JavaScript
  4. チョコクランチ
